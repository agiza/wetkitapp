-- SUMMARY --

Upper Ribbon shown on every page of our Drupal 7 site.

-- REQUIREMENTS --

*  Jquery Colorpicker.

-- INSTALLATION --

* Install as usual, copy and extract ribbon.zip in the folder '/sites/all/modules'

-- FEATURES --

* Background Color Customizable. 
* Font Color Customizable.
* Allow Background Transparence.
* ( Show | Hidde ) Close Button.
